Clculo de tamao de cristalito a partir de la formula de Debye-Scherrer

$$D = \frac{K \lambda}{\beta cos(\theta)}$$

Donde:

  • K: Factor de estructura (0.89 para cbicas [2])
  • $\lambda$: Longitud de onda CuK$\alpha$ (1.5406 )
  • $\beta$: Distancia entre Punto A y Punto B
  • D: Tamao de cristalito

def calc_tamanio_cristalito_scherrer (beta, theta):
    K = 0.89 # Para cbicas segn la referencia
    LAMBDA = 1.5406 #Longitud de onda Cobre K alfa
    
    #print(theta)
    theta_rads = np.deg2rad(theta)
    angulo = np.cos(theta_rads)
    #print(angulo)
    cristalito_size = (K * LAMBDA)/(beta * angulo)
    #print(f'Tamao de cristalito calculado: {cristalito_size.values[0]} ')
    return cristalito_size.values[0] * 0.1 # Se multiplica por 0.1 para convertir A a nm

Calculo del cristalito utilizando la frmula de Williamson-hall

Para calcular el cristralito a travs de este mtodo tenemos que utilizar la siguiente expresin

$$ \beta_{hkl}cos\theta = \frac{K\lambda}{D} + 4\epsilon sen\theta$$

si observamos bien podemos darnos cuenta que la frmula adopta la forma: $$ y = mx+b $$

Haciendo la regresin lineal de nuestros datos podemos obtener la siguiente expresin

$$ y = 0.11535040977386593 x + 0.24479779320473088 $$

por lo que podemos decir que:

$$ \frac{K\lambda}{D} = 0.24479779320473088 $$

Asumiendo que:

  • K = 0.89
  • $\lambda$ = 1.5406

Podemos despejar y obtener el resultado, tal que:
